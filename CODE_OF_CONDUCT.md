# Code of Conduct

## Our Commitment

CareChain is an open-source healthcare technology project. We are committed to creating a respectful, professional, and inclusive environment for contributors, users, and maintainers.

Healthcare software requires trust, care, and responsibility. Everyone contributing to CareChain is expected to communicate with patience, clarity, and respect.

## Expected Behavior

Contributors are expected to:

- Be respectful and constructive.
- Use welcoming and professional language.
- Give and receive feedback gracefully.
- Focus discussions on the project and its users.
- Be careful when discussing healthcare, privacy, security, and patient data.
- Respect different experience levels and backgrounds.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or behavior.
- Sharing private information without permission.
- Publishing secrets, credentials, or sensitive healthcare data.
- Making false claims about CareChain, ABDM, ABHA, government approval, or medical capabilities.
- Using issues or pull requests for spam or unrelated promotion.

## Healthcare And Privacy Responsibility

CareChain may involve healthcare-related workflows. Contributors must never upload, publish, or request real patient records, Aadhaar details, ABHA credentials, or personal health information in public discussions, issues, pull requests, screenshots, or demos.

Use mock data only.

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or block contributors who violate this Code of Conduct.

## Reporting

If you see behavior that violates this Code of Conduct, report it privately to the project maintainer.
