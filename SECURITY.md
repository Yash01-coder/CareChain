# Security Policy

CareChain is an open-source healthcare technology project. Security, privacy, and trust are core priorities.

## Project Status

CareChain is currently in prototype-to-MVP development.

It should not be used with real patient data in production until security review, privacy review, deployment hardening, and compliance checks are completed.

## Reporting A Security Issue

If you discover a security issue, please do not open a public GitHub issue with sensitive details.

Instead, contact the project maintainer privately.
