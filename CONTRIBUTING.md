# Contributing to CareChain

Thank you for your interest in contributing to CareChain.

CareChain is an India-first, open-source digital healthcare platform focused on patient-owned medical records, consent-first sharing, blockchain-based integrity verification, and future AI-assisted healthcare workflows.

## Project Status

CareChain is currently moving from prototype to MVP.

Current focus:

- Repository cleanup
- v2.0 MVP planning
- UI/UX foundation
- Backend security improvements
- India-first ABDM/ABHA-style planning
- Blockchain record integrity improvements
